## How to use

### Back-end Code: Laravel

- Clone the repository with __git clone__
- Run `php artisan serve` and enter the URL shown in the Terminal.


### Front-end Code: Vue.js

- Clone the repository with __git clone__
- Run `npm run serve` and enter the URL shown in the Terminal.
